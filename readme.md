# Edward Bank CLI

A simple Java console application that simulates basic banking operations.

## Features
- Create account with initial balance  
- PIN authentication required for transactions  
- Mandatory PIN change on first use  
- Deposit and withdraw money  
- Account activation and deactivation  
- Account lock after multiple incorrect PIN attempts  

## Technologies
- Java
- Object-Oriented Programming (OOP)
- Git & GitHub

## Purpose
This project was built to practice object-oriented programming, application flow design, and basic security logic for backend development.
